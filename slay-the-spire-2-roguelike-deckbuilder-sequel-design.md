# 『Slay the Spire 2』は何を守り、何を壊したのか――ジャンルを定義したデッキ構築ローグライクの続編設計

> **注記**：本稿は2026年7月17日時点の早期アクセス版に基づく。今後の開発と更新により、カード、敵、バランス、協力プレイなどの内容は変更される可能性がある。[[1](#ref-1)]

2017年に早期アクセスへ登場した『Slay the Spire』は、カードを選んでデッキを育てながら、失敗のたびに最初から塔へ挑む構造を広く定着させた作品である。開発元Mega Crit自身も、続編の告知文で前作をジャンルを定義した作品として位置付けている。[[1](#ref-1)] 前作は2019年1月に1.0へ移行した。[[2](#ref-2)]

その続編『Slay the Spire 2』は、2026年3月5日にSteam早期アクセスを開始した。2026年7月17日現在、1.0正式版には到達しておらず、カード、イベント、環境、敵などを早期アクセス中に追加・調整する方針が公式に示されている。つまり本作は、完成済みの「正解」を鑑賞する対象ではなく、ジャンルの基礎を残したまま何を作り替えるかを、プレイヤーとともに検証している最中の続編である。[[1](#ref-1)]

本稿が扱うのはローグライク一般の定義ではない。ローグライクという言葉の射程や系譜は「[結局ローグライクってなんなのさ？](what-is-a-roguelike.md)」に譲り、ここでは単一タイトルの続編が、既存プレイヤーの身体化した判断をどう保ち、どこへ新しい判断を足したかを分析する。

***

## 1. 前作が残した「一周の問い」

『Slay the Spire』の一周は単純である。経路の分岐する塔を上り、戦闘・店・イベント・休憩地点を選び、戦闘報酬のカードやレリックでデッキを変え、ボスを越えられなければ次のランで別の選択を試す。ターン制カード戦闘とランごとに変わる入手物を結び、事前に完成したデッキを持ち込むのではなく、 **そのランで提示された材料から解法を組み立てる** ことを中心に置いた。開発者Anthony Giovannetti氏のGDC講演でも、作品を「ローグライクとデッキビルダーの組み合わせ」と説明している。[[3](#ref-3)]

この型で重要なのは、カードを多く取ることではない。次の敵、残りHP、手持ちの防御、将来のボス、除去の機会を見て、「今の一枚」がデッキ全体の事故率と勝ち筋をどう変えるかを判断することだ。ランダム性は正解を隠すためにあるのではなく、毎回同じ強カードの手順へ収束することを防ぎ、状況判断を作るためにある。

したがって続編の設計で最初に守るべきなのは、カード名や敵名そのものではない。プレイヤーが戦闘、報酬、経路選択を往復し、敗北を次の仮説へ変換できる因果である。続編も塔を登り、戦闘でカードを使い、報酬でデッキを更新し、再挑戦する。この骨格を大きく崩していないことが、以下の更新を「別ゲーム」ではなく「続編」として読める土台になっている。[[1](#ref-1)]

```
# 仕上げAIへ図表制作依頼
『Slay the Spire』シリーズの基本ループを示す循環図を作成してください。「塔を上る」→「戦闘」→「カード・レリック報酬でデッキを更新」→「店・イベント・休憩地点を選ぶ」→「ボスに挑む（敗北した場合は次のランへ）」という5つの要素を円環状に配置し、失敗から次のランへの学習が生まれる構造が視覚的にわかるようにしてください。本文第1章の「一周の問い」という表現に対応する図です。
```

***

## 2. 維持したもの：既存の知識を捨てさせない

早期アクセス開始時点のプレイアブルキャラクターは5人である。前作からアイアンクラッド、サイレント、デフェクトが続投し、新たにネクロバインダーとリージェントが加わった。前作に慣れたプレイヤーは、自己ダメージと回復、毒・短剣・捨て札、オーブといった既存三人の発想を入口にできる。[[1](#ref-1)][[4](#ref-4)]

これは単なる懐古ではない。続編の初期数時間で、プレイヤーに新キャラクター、新敵、新報酬、新UIを一度に学ばせると、負けた原因を分類できなくなる。既知の開始デッキ、見慣れたカード群、塔を登る判断を足場として残せば、失敗を「新敵の性質」「新しいカードの評価」「経路の選び方」に切り分けやすい。PC Gamerの実プレイ記事も、続投三人の開始デッキと開始レリック、カードプールの多くが前作と親しみ深いと報じている。[[5](#ref-5)]

ここでの学びは、続編の継続性を「見た目が似ているか」で測らないことにある。プレイヤーが既に持つ判断規則をどこまで再利用できるかを先に設計し、その上で新規性を重ねるべきである。

***

## 3. 更新したもの：同じ選択密度に別の軸を足す

### キャラクターは「追加」ではなく、既存の読み替えである

新規のネクロバインダーは使い魔Osty、魂、Doomを扱い、[[6](#ref-6)] リージェントはStarsと専用武器Sovereign Bladeを軸にする。[[7](#ref-7)] いずれも新しいリソースを持ち、前作の三人とは異なる順序で手札と戦闘を組み立てる。これはカード枚数を増やすだけでなく、「何を貯め、いつ消費するか」というキャラクター固有の問いを増やす更新である。

```
# 仕上げAIへ画像引用依頼
Mega Crit公式ニュースレター（2025年10月号または2025年12月号）に掲載されている、ネクロバインダーまたはリージェントの公式ビジュアル画像を、本文のこの段落への引用候補として検討してください。著作権法上の引用要件（必要な範囲・主従関係・出典明記・無改変）を満たす形での使用を前提とし、満たせない場合はスキップしてください。
```

一方、続投キャラクターも保存版ではない。開発元は、前作のカードを一部外し、新カードを加え、古いカードにも調整を施す方針を明かしている。サイレントの新キーワード Sly は、その代表例だ。Sly を持つカードは捨てられた際に無料で自動使用されるため、従来なら手札の質を整える補助だった捨て札が、直接の行動回数やダメージへ接続する。[[8](#ref-8)]

この変更は、サイレントの「捨て札」という既知の語彙を残しながら、評価基準を変える。例えば、捨て札カードは不要札を処分するだけでなく、Sly の発火装置にもなる。プレイヤーは一枚単体の効率ではなく、手札を動かす順番、無料使用の誘発、山札を一巡させる速度を同時に考える必要がある。既存キャラクターの再設計では、古いアーキタイプを完全に捨てるより、 **慣れた言葉の意味を少しずらす** 方が、学習コストを制御しながら再発見を作りやすい。

### 報酬と表現は、選択の見え方を変える

続編には、ラン中にカードへ付与される Enchantment がある。これはカードに追加効果を与える修正で、そのランの間だけ持続する。前作の通常のカード強化とは別に、同じカードでも「今回のランでは何を担うか」を変えられるため、カード評価を固定表から状況依存へ戻す仕組みになっている。[[9](#ref-9)]

また、ボスレリックに相当する位置にはAncientsが提示され、各Actの入口で強力な恩恵を選ぶ。恩恵には大きな代償を伴うものもある。単に次の戦闘を強くする報酬ではなく、以後の経路やデッキ構成へ制約を置く報酬は、選択を「強いか弱いか」から「どの不自由を引き受けるか」へ変える。[[10](#ref-10)]

技術面では、開発元がGodotへの移植と安定化、より拡張しやすい設計、アートとUIの刷新を続編の理由として挙げている。演出の滑らかさを単独で目的化するのではなく、敵・選択肢・状態変化を読み取らせる情報設計と、今後の追加を受け止める制作基盤を同時に更新した点に意味がある。[[11](#ref-11)]

***

## 4. 型を壊す更新：4人オンライン協力プレイ

最大の断絶は、最大4人のオンライン協力プレイである。前作の基本ループは一人で考え、一人で失敗を引き受ける設計だった。続編はそのランを共有する。ただし、全員で一つのデッキを回すのではない。各プレイヤーは自分のエネルギー、カード、レリック、所持金を持ち、敵の能力値は人数に応じて変化する。敵の攻撃はパーティ全体へ届き、全員がターン終了を選んで次へ進む。[[12](#ref-12)]

```
# 仕上げAIへ図表制作依頼
『Slay the Spire 2』の協力プレイの構造を示す図を作成してください。中央に「敵（能力値は人数に応じて変化、攻撃はパーティ全体に届く）」を置き、その周囲に2〜4人のプレイヤーを配置してください。各プレイヤーが自分専用のエネルギー・カード・レリック・所持金を持つことを示しつつ、敵からの攻撃だけがパーティ全体で共有されるという「個別資源×共有される脅威」の非対称な構造が一目でわかるようにしてください。
```

ここで協力プレイを「難易度を下げるための人数増加」と扱わなかったことが重要である。協力専用カードが報酬に出現し、味方の状態を見ながらカードを使う。例えば一人が敵へ脆弱などのデバフを付与し、次の味方が大きな攻撃を重ねるなら、カードの価値は単独プレイのダメージ量ではなく、パーティ全体の行動順と手札に依存する。ゲーム内では味方の手札、エネルギー、カーソル位置が見え、誰が何を考えているかを最低限共有できる。[[12](#ref-12)][[13](#ref-13)]

この構造は、デバフをかける人と追撃する人という「役割」を固定するためのものではない。次の手番で誰が何を出せるかに応じて、その場で役割を組み替えるための設計である。一人用ではカードAの後にカードBを出す順序だけを考えればよかった。協力プレイでは、カードAを誰が出し、その効果を誰のカードBで回収し、全員が危険な敵ターンを越えられるかまでが一つの選択になる。

同時に、協力には待ち時間、経路投票、報酬の競合という摩擦がある。本作は経路投票や宝箱の競合をゲーム内の手の表現と抽選で処理し、協議が必要な部分を消さずにテンポの破綻を抑えようとしている。協力要素の価値は、戦闘火力を足すことではなく、単独プレイにはなかった「他者の情報を読んで順序を組む」判断を作れるかにある。[[13](#ref-13)]

## 5. 早期アクセスは、バランス調整そのものを公開する

早期アクセスは未完成の言い換えではない。カードゲームでは、カード単体、組み合わせ、敵、報酬、難易度が連鎖するため、調整の内容と伝え方までが運営設計になる。

2026年3月19日（北米時間。日本時間では3月20日）公開のベータパッチ v0.100.0 は、無限コンボを成立しにくくすることを主眼に、大規模な変更を行った。これは正式版への強制更新ではなく、公開ベータブランチを選んだプレイヤー向けの調整だった。それでも更新直後にはSteamで多数の否定的レビューが集まり、Mega Critは早期アクセスでの進行は直線的ではなく、変更が恒久的とは限らないと説明した。[[14](#ref-14)][[15](#ref-15)]

ここで見るべきは、反発を「プレイヤーが変化を嫌った」と片付けないことである。無限コンボは、長い試行の末に発見した達成感、既存カードの評価、上達の証明と結び付いている。開発側がそれを難しくすれば、挑戦性やカード選択の多様性を守れる可能性がある一方、プレイヤーには自分が覚えた解法を後から取り上げられたように映る。しかもパッチノートの一項目は、デッキ全体の相互作用を読むプレイヤーほど大きな意味として受け取る。

プランナーがここから学べるのは、数値を変える前に「何を守るための変更か」を説明し、観測期間、対象ブランチ、戻す条件を明確にすることだ。無限コンボを減らすなら、想定外の即死や単調な最適解を抑えたいのか、ボス戦で別の勝ち筋を要求したいのかを分けて語る必要がある。早期アクセスのフィードバック窓口をゲーム内にも用意するとした公式方針は、レビューだけに不満が集約されない導線を作ろうとする点で重要である。[[1](#ref-1)]

```
# 仕上げAIへ図表制作依頼
『Slay the Spire 2』の早期アクセス後の出来事を追う簡潔な時系列図を作成してください。「2026年3月5日：早期アクセス開始」→「2026年3月19日（日本時間3月20日）：ベータパッチv0.100.0公開（無限コンボ対策が主眼）」→「パッチ直後：Steamで否定的レビューが急増」→「Mega Critが早期アクセスの進行方針を説明」という4段階を横並びの矢印で示してください。
```

## 6. 続編を設計するときの切り分け

『Slay the Spire 2』の現時点の設計からは、ジャンルを定義した作品の続編における切り分けが見える。

| 切り分ける対象 | 守るべきもの | 壊してよいもの | 確認すべき問い |
| --- | --- | --- | --- |
| 基本ループ | 塔を進み、提示された材料でデッキを編集し、再挑戦から学ぶ因果 | 敵、報酬、経路の具体的な内容 | 失敗の理由を次のランの仮説に変えられるか |
| 既存キャラクター | 役割を理解する入口と既存の語彙 | カード群、誘発条件、強い組み合わせ | 旧知識は助けになるか、それだけで最適化できてしまわないか |
| 新規性 | 一人でも成立する戦略判断 | 協力、専用カード、複数人の行動順 | 人数増加が火力増加で終わらず、新しい選択を生むか |
| 調整運営 | 挑戦の意味、判断の可読性 | 個別カードの性能、発見済みの最適解 | 変更理由と検証範囲を、体験の言葉で伝えられるか |

続編で「型を守る」とは、前作のカードを複製することではない。プレイヤーがなぜ失敗し、何を次回へ持ち帰るかという学習の骨格を守ることである。「型を壊す」とは、目新しい要素を積むことでもない。4人協力のように、既存の勝ち筋評価だけでは足りず、他者の手札、順番、危険を読む新しい判断を作ることである。

早期アクセスでは、この二つの境界が固定されない。現段階の Sly や協力専用カード、無限コンボ対策が1.0でどう変わるかは決まっていない。だからこそプランナーは、数値の変更を「弱体化」だけで評価せず、基本ループの学習可能性、多様な勝ち筋、プレイヤーが発見を自分のものと思える感覚を同時に観測すべきである。

『Slay the Spire 2』は、ジャンルを定義した前作の答えを保存するのではなく、答えを出すための問いを保存しようとしている。1.0正式版で、その問いと協力プレイ、継続調整がどのように一本の体験へ着地するかは、今後の更新で判断すべき課題である。

```
# 仕上げAIへ図表制作依頼
本文第6章の表（切り分ける対象／守るべきもの／壊してよいもの／確認すべき問い）をインフォグラフィックとして視覚化してください。「基本ループ」「既存キャラクター」「新規性」「調整運営」の4項目を並べ、各項目について「守るべきもの」（安定を示す色）と「壊してよいもの」（変化を示す別の色）を対比的に配置し、表の内容がひと目でわかるようにしてください。
```
